## Sobre o Projeto

Ecommerce desenvolvido para digitalizar a fábrica de acessórios de moda do meu pai. Substitui o processo manual de vendas por WhatsApp por um sistema web completo com catálogo, carrinho e painel administrativo.

## Funcionalidades

- Catálogo de produtos com busca e filtros
- Carrinho de compras e sistema de pedidos
- Autenticação JWT (login/registro)
- Painel admin para gestão de produtos, pedidos e clientes
- Controle de estoque

## Stack Tecnológica

**Frontend:** React, TypeScript, Vite, Tailwind CSS, Axios  
**Backend:** Node.js, Express, Prisma, MySQL, JWT, bcrypt  
**DevOps:** Docker, Docker Compose

## Destaques Técnicos

- Arquitetura MVC com separação clara de responsabilidades
- Prisma ORM para MySQL com migrations e type-safety
- Autenticação JWT com cookies HttpOnly seguros
- TypeScript em todo o projeto (frontend e backend)

## Impacto

Projeto **em desenvolvimento** para fábrica do meu pai, tem o objetivo de substituir o processo manual e trazer organização ao negócio familiar.
