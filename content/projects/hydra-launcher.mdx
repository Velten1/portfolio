## Sobre o Projeto

Contribuição open-source para o **Hydra Launcher**, um launcher de jogos multiplataforma. Implementei a funcionalidade de importação automática da biblioteca Steam, permitindo que o launcher monitore e analise a pasta da Steam para detectar e importar jogos automaticamente.

## Funcionalidades

- Monitoramento automático da pasta de instalação da Steam
- Detecção e análise de jogos instalados
- Importação automática de jogos para o Hydra Launcher
- Integração com a biblioteca Steam do usuário
- Sistema de watch para detectar mudanças em tempo real

## Stack Tecnológica

**Tecnologias:** Electron, TypeScript, Node.js  
**APIs:** Steam API, File System Watchers  
**Plataforma:** Desktop (Windows, macOS, Linux)

## Destaques Técnicos

- Implementação de file watchers para monitoramento de diretórios
- Análise de arquivos de configuração da Steam
- Integração com sistema de importação do Hydra Launcher
- Tratamento de erros e edge cases
- Código seguindo padrões do projeto open-source
